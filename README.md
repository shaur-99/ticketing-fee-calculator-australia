# Ticketing Fee Calculator Australia

A one-page static calculator for comparing Australian ticketing platform fees.

The calculator lets a vendor enter:

- number of paid tickets
- ticket price in AUD

It then compares the money taken by:

- Your app: 1.95% + A$0.99 per paid ticket
- Eventbrite Australia: 5.35% + A$1.19 per paid ticket
- Humanitix standard: (4% + A$0.99) plus 10% GST
- Luma free plan: 5% platform fee + Stripe AU 2.9% + A$0.30

The bottom profit metric calculates tickets sold multiplied by A$1.43.

Assumptions: one ticket per order, domestic Australian card, organiser absorbs fees where comparable.
